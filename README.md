# Customer Login & Registration Form

This is a simple web project that demonstrates a **login and registration system** with basic **JavaScript form validation**.

# Features

* Login form with:

  * ID validation (cannot be empty)
  * Password validation (minimum 8 characters)

* Registration form UI

* Styled using CSS (flexbox layout)

* Alert-based user feedback

# Technologies Used

* HTML
* CSS
* JavaScript

#Project Structure

```
form-validation-js/
│── another.html

│── scripting.js
│── README.md
```

# How It Works

* User enters ID and password
* JavaScript checks:

  * If ID is empty
  * If password is at least 8 characters
* Shows alert messages based on validation
